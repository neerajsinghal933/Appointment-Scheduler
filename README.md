## 🚀 Getting Started

## 🔧 Activate Virtual Environment
 💻 On macOS / Linux:
```bash
source venv/bin/activate
```

### 🛠️ Install Dependencies

Make sure you have Python 3 and `virtualenv` installed.

```bash
pip install -r requirements.txt
```

## ▶️ Run the Flask App
```bash
flask run
```

Then open your browser and go to:
http://127.0.0.1:5000/
